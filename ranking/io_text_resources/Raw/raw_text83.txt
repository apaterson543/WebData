Information</h3>\n<h4>All Travelers</h4>\n<p><b>October 1, 2020</b> \xe2\x80\x93 Click on this link for <a rel="noopener noreferrer" href="https://hawaiicovid19.com/travel/#travel-FAQs" target="_blank">Hawai\xe2\x80\x98i Travel FAQs</a>, which includes information on Hawai&#8217;i&#8217;s Pre-Travel Testing Program. &nbsp;</p>\n<p><strong>Sept 23, 2020</strong> \xe2\x80\x93 The State of Hawaii continues to enforce the 14-day mandatory quarantine requirements for out-of-state travelers.&nbsp; There is an inter-island travel quarantine only for travelers arriving in the counties of&nbsp; Kaua&#8217;i, Hawai&#8217;i, Maui and Kalawao (Kalaupapa), with updates to the self-quarantine exceptions.&nbsp; </p>\n<p>The Governor&#8217;s <a href="https://governor.hawaii.gov/wp-content/uploads/2020/09/2009139-ATG_Thirteenth-Supplementary-Proclamation-for-COVID-19-distribution-signed.pdf">thirteenth supplementary emergency proclamation</a> identifies the Rules Relating to COVID-19 Health Screening Process and Travel Self-Quarantine, referred to ask &#8220;Travel Rules&#8221;.&nbsp; A negative COVID-19 test exception shall become effective October 15, 2020. &nbsp;</p>\n<p><strong>Sept 1, 2020</strong> \xe2\x80\x93 The State of Hawaii traveler information is now captured in digital format.&nbsp; All travelers are highly recommended to use the State\xe2\x80\x99s &nbsp;<a href="https://travel.hawaii.gov/">Safe Travels program</a>&nbsp;<strong>before</strong> arriving.&nbsp; Upon arrival, all travelers that use the program will have to show their Safe Travel\xe2\x80\x99s QR code to the State screener and will be processed quicker.&nbsp; Those that arrive without a QR code will need to create a Safe Travels
